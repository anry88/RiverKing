# Branding Assets

This directory contains generated Android store assets for RiverKing.

Current files:

- `android-icon-1024.png` — square store icon source for itch.io and Google Play
- `itch-cover-1280x720.png` — cover image for the itch.io page
- `play-feature-1024x500.png` — Google Play feature graphic

These files are generated by:

- [mobile/android-app/scripts/generate_brand_assets.py](/Users/hq-k14lcdcq7d/Documents/IdeaProjects/RiverKing/mobile/android-app/scripts/generate_brand_assets.py)

The generator also refreshes the launcher icon layers used by the Android app:

- `mobile/android-app/app/src/main/res/drawable-nodpi/ic_launcher_background.png`
- `mobile/android-app/app/src/main/res/drawable-nodpi/ic_launcher_foreground.png`

Regenerate everything after a branding change:

```bash
python3 mobile/android-app/scripts/generate_brand_assets.py
```
